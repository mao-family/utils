# @mao-family/utils

Utility scripts and tools for Mao Family automation.

## Stack

- TypeScript (ES2022, strict)
- Vitest for testing
- ESLint + Prettier for code quality

## Commands

- `npm run build` — compile TS
- `npm test` — run tests
- `npm run lint` — lint
- `npm run format` — format code

## Rules

- All code in `src/`, tests in `tests/`
- Each utility script should be self-contained with clear exports
- Write tests for all non-trivial logic
